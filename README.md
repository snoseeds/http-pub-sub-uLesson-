## Installation Requirements

* Node js
* Typescript
* npm
* Git
* Docker

### Installation code
run: 
```Bash
    $ git clone https://github.com/snoseeds/http-pub-sub
    $ cd natterbase-test
    $ npm run start:dev
    $ npm install
    $ npm start
```

### Tests
```Bash
    $ npm test /Set MODE=TEST in src/config/config.service.ts before that
```
